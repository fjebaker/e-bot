EGameFactory¶

This class defines the EGameFactory super-class, used to define new interactive game components. abstracts.EGameFactory defines a number of attributes, which should be defined by the subclasses.

@execute_rounds(max_rounds: int = 0, prompt_continue: bool = True)¶

Decorator method for games with multiple rounds, implementing optional checks at the end of each round as to whether player want to continue with same players, poll for new players or stop playing completely. Note that this will throw an error if max_rounds is set to 0 and prompt_continue is set to False, as this would represent infinite rounds with no way to stop.

Parameters

  • max_rounds – the maximum number of rounds to play. If set to 0, will have no maximum. optional, defaults to 0

  • prompt_continue – whether to prompt users to continue. optional, defaults to True
